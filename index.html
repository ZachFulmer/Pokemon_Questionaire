Who's That Pokemon

So, you think you are a pokemon Master?

Game Rules!

  • You will be given a Pokémon card to try and guess it's name.
  • You're given four possible answers to choose from.
  • Each incorrect answer results in losing a life.
  • For Each correct answer player is awarded 10 points.
  • The game ends when the player's lives reaches zero.
  • Objective of the game is to get the highest score possible.
